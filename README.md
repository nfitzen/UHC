# UHC
The popular Minecraft minigame recreated in a datapack.

## Features to be

I made a version of this pretty early on (04/10/2019), but I'm rewriting it from scratch (ish) to include:

- A setup book with customization options upon `reload`
    - If not that, then at least a decent CLI
- Optimized and cleaner code
    - More usage of advancements and loot tables for, e.g., death detection
    - Using `schedule` for stuff like code running every second
- Addon packs for memes, crafting recipes, and possible weird stuff.
